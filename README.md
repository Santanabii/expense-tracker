# expense-tracker
## PROJECT DESCRIPTION ##
The Expense Tracker is a web-based application that allows users to manage their daily expenses efficiently. Users can add expenses with a description, amount, and category, view all recorded expenses, filter them by category, and calculate the total expenses for the day. The application also stores expense data in localStorage so that information remains available even after refreshing the page.
This project demonstrates the use of Object-Oriented Programming (OOP), functional programming concepts, and modern JavaScript features such as destructuring, spread and rest operators. Unit testing is implemented using Jest to ensure that the core functionalities work correctly.

## Features ##
-Add a new expense with a description, amount, and category
-View all expenses in a structured list
-Delete an expense from the list
-Filter expenses by category
-Automatically calculate total daily expenses
-Store expenses using localStorage for persistence
-Unit testing for important functions

## Technologies Used ##
-HTML – Structure of the application
-Tailwind CSS – Styling and responsive UI using Tailwind CSS
-JavaScript – Core application functionality
-localStorage – Persistent data storage in the browser
-Jest – Unit testing framework
-Git – Version control
-GitHub Pages – Project deployment using GitHub Pages
